This directory holds your project images.

Naming convention:
  - Use lowercase, hyphens, descriptive names
  - Example: tidytuesday-penguin-sizes.png
  - Example: d3-transit-map-interactive.png

Recommended:
  - Export ggplot2 at 1200x800px minimum
  - Use PNG for static plots, SVG where possible
  - For thumbnails in the grid, images will be cropped to ~320x220px
    so make sure the important content is centered
